// Apa itu Monte Carlo Simulation?

Monte Carlo adalah metode simulasi yang menjalankan skenario ribuan kali dengan input acak, lalu melihat distribusi hasilnya. Karena kita tidak bisa memprediksi satu pertandingan secara pasti, kita simulasikan 10.000 kemungkinan turnamen dan hitung berapa kali setiap tim menang.

// Model
# Setiap pertandingan:
score_A = strength_A + gauss(μ=0, σ=0.15)
score_B = strength_B + gauss(μ=0, σ=0.15)
winner = A if score_A > score_B else B

# σ = 0.15 → upset rate ~30%, realistis

Strength rating dikonversi dari FIFA Ranking resmi April 1, 2026. France #1 → 0.93, New Zealand #88 → 0.52. Gaussian noise mewakili faktor tak terduga: cedera, momentum, keberuntungan.

// Kenapa Spain / France Favorit?

Ini efek grup yang menarik — mereka tidak bertemu sampai babak knockout lanjut. Spain di Grup H bersama Cape Verde, Saudi Arabia, Uruguay. France di Grup I (Group of Death) bersama Senegal, Norway, Iraq. Tapi begitu lolos grup, bracket terbuka lebar.

⚠ Tim yang tidak lolos ke WC 2026:
Italy (gagal playoff UEFA untuk ketiga kalinya — kalah adu penalti ke Bosnia), Poland (kalah dari Sweden), Denmark (kalah dari Czechia). Ini storyline terbesar turnamen ini.
// Law of Large Numbers

Coba jalankan 1.000 vs 50.000 simulasi dan bandingkan hasilnya. Semakin banyak N, semakin stabil probabilitasnya — ini adalah Law of Large Numbers in action.
